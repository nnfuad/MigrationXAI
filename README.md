# MigrationXAI

This project demonstrates the factors driving irregular migration of Bangladeshi migrants through the Mediterranean Sea. It uses a synthetic dataset to train a machine learning model that predicts the likelihood of irregular migration based on various socio-economic and environmental factors and use Explainable AI (XAI) techniques to interpret the model's predictions. As real data on irregular migration is often scarce and sensitive, this project serves as a proof of concept for using synthetic data to understand and predict migration patterns.
